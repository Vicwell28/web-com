El Modelo de Caja (Box Model) de CSS

El Modelo de Caja es un concepto esencial en CSS que describe cómo se estructuran y calculan los elementos en una página web. Cada elemento HTML se representa como una caja rectangular compuesta por cuatro partes:

  • Contenido (Content): Es el contenido real del elemento, como texto, imágenes, etc.
  • Relleno (Padding): Espacio interno entre el contenido y el borde.
  • Borde (Border): Línea que rodea el relleno y el contenido.
  • Margen (Margin): Espacio externo que separa el elemento de otros elementos.

Propiedad box-sizing

La propiedad box-sizing controla cómo se calculan las dimensiones de un elemento. Tiene dos valores principales:

  • content-box (predeterminado): Las propiedades width y height sólo incluyen el contenido. El relleno y el borde se agregan fuera de estas dimensiones.
  • border-box: Las propiedades width y height incluyen el contenido, el relleno y el borde. Esto facilita el manejo de las dimensiones totales del elemento.
